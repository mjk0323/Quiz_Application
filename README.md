: quizs 폴더 안의 quiz 파일만 수정하면 됨, quiz.js를 메모장으로 열어서 클로드가 준 코드로 대체

### ⌨️클로드 프롬프트
- 지정할 것 : 문제 유형(객관식, 단답형, 서술형), 문제 개수, 시험 출제 스타일, 객관식은 정답 선지가 티나지 않도록, 중요한 부분이 어디인지, 만약 퀴즈를 한 번 완료했다면 틀린 문항 번호
- 중요한 부분이 어디인지 지정할 때 
  <br>텍스트 파일일 때 : 
  <br>^중요한내용강의내용설명설명^ 다른 내용 이렇게 표시해두고 ^ 이 기호 사이에 있는 건 중요한 부분이라고 지정해주면 됨.
  <br>ppt나 pdf 파일일 때 : 
  <br>밑줄쳐진 부분, 굵은 글씨, 기울어진 글씨라고 지정해주면 됨
- 시험 출제 스타일 지정 시 교수님이 이전에 출제하셨던 퀴즈, 시험문제가 있으면 넣어주고 이 파일은 교수님이 이전에 내셨던 퀴즈/시험인데 이 파일을 자세히 분석해서 이 유형과 똑같이 시험 출제 해줘라고 하면 됨
- 객관식, 단답형, 서술형 말고 다른 문제 유형 지정하고 싶으면 원하는 스타일의 문제 유형 예시 갖다넣고 똑같이 해달라고 하면 됨.
- ppt 파일 pdf 파일 넣으면 체감상 토큰이 더 빨리 닳아서 Gemini나 GPT한테 파일 넣고 텍스트 추출해달라한 다음에 텍스트만 복붙해서 붙이는게 나았음. 텍스트로 넣으면 굵은 글씨, 밑줄 다 없어지니까 Gemini, GPT에게 텍스트 추출 명령 시 굵은 글씨, 밑줄은 *내용* 이런 식으로 표시해달라고 하면 됨.

### ➕참고 :

### 📎프롬프트 입력 예시

첨부파일 : [운영체제01.pdf, 운영체제02.pdf, 운영체제03.pdf, quiz.js]

이 프롬프트에 맞춰서 문제 내줘

수준 : 대학교 졸업학년 수준<br>
과목 분류 : 전공수업<br>
출제 : 영어, 한글 (영어로 적고 옆에 한글 해설 제공)<br>
답안 : 한글, 영어 둘 다 가능<br>
범위 : 운영체제01.pdf, 운영체제02.pdf, 운영체제03.pdf<br>
난이도 : 기초, 기본, 이해, 활용까지 전범위. 기본적인 개념 질문은 물론 모든 텍스트를 아울러서 연결할 수 있는 내용은 연결시키고 말을 바꿔서 모호하게 낼 수 있는 문제는 그렇게 내줘. 변별력있고 공부한 대학교 4학년 학생을 가를 수 있는 난이도로. 상대평가.<br>
문제 개수 : 모든 범위에서 40문제<br>
문제 유형 : 객관식, 단답형 문제 (단답형이라 함은 어떤 개념을 쓰시오 했을 때 딱 단어만 아니면 해당하는 개념만 쓰는걸 말하는거고 서술하시오 이런거 안됨. 계산 문제 같은건 괜찮음<br>
템플릿 : quiz.js (다른 파일은 정의되어 있어서 quiz.js만 수정하면 됨, 절대 quiz.js의 문제를 사용하지 않아야함. quiz.js는 템플릿용임)<br>
