💁사용법
: 다 저장하고 실행 시 index.html을 브라우저로 열면 됨

✍️문제 수정
: quizs 폴더 안의 quiz 파일만 수정하면 됨, quiz.js를 메모장으로 열어서 클로드가 준 코드로 대체

⌨️클로드 프롬프트
: 강의 내용 파일(또는 텍스트 정리본) + quiz.js 파일 + 이 문제들을 내가 준 강의 파일의 내용을 써서 문제 출제해줘, 절대 quiz.js의 내용을 쓰지 않아야함. 내 시험범위는 [강의 파일 제목]이야 (e.g. 내 시험범위는 bsd02.pptx, bsd03.pptx, bsd04.pptx 파일만 해당돼)
- 지정할 것 : 문제 유형(객관식, 단답형, 서술형), 문제 개수, 시험 출제 스타일, 객관식은 정답 선지가 티나지 않도록, 중요한 부분, 만약 퀴즈를 한 번 완료했다면 틀린 문항 번호

📎프롬프트 입력 예시

[운영체제01.pdf, 운영체제02.pdf, 운영체제03.pdf, quiz.js]

이 프롬프트에 맞춰서 문제 내줘

수준 : 대학교 졸업학년 수준
과목 분류 : 전공수업
출제 : 영어, 한글 (영어로 적고 옆에 한글 해설 제공)
답안 : 한글, 영어 둘 다 가능
범위 : 운영체제01.pdf, 운영체제02.pdf, 운영체제03.pdf
난이도 : 기초, 기본, 이해, 활용까지 전범위. 기본적인 개념 질문은 물론 모든 텍스트를 아울러서 연결할 수 있는 내용은 연결시키고 말을 바꿔서 모호하게 낼 수 있는 문제는 그렇게 내줘. 변별력있고 공부한 대학교 4학년 학생을 가를 수 있는 난이도로. 상대평가.
문제 개수 : 모든 범위에서 40문제
문제 유형 : 객관식, 단답형 문제 (단답형이라 함은 어떤 개념을 쓰시오 했을 때 딱 단어만 아니면 해당하는 개념만 쓰는걸 말하는거고 서술하시오 이런거 안됨. 계산 문제 같은건 괜찮음
템플릿 : quiz.js (다른 파일은 정의되어 있어서 quiz.js만 수정하면 됨)
형식 : 맞춘 문제, 틀린 문제를 한 눈에 볼 수 있도록. 다 풀고나면 틀린 문항 번호와 어떤 부분을 공부해야하는지 띄워주기. 객관식은 답을 클릭하면 바로 정답이 뭔지 띄우고 정답과 함께 설명도 넣기. 단답형도 마찬가지. 모든 문제는 채점이 되어야함. 또는 적합할 것 같은 기능
